CREATE PROC [core].[RetrieveAssetCollectionByInstalledSoftware]
   @SoftwareName NVARCHAR(256) ,
   @Start INT = 1 ,
   @Limit INT = NULL ,
   @IsAscending BIT = 1 ,
   @TotalCount INT OUTPUT
AS
   BEGIN
       DECLARE @End INT;
   
       SELECT  @TotalCount = COUNT(1)
       FROM    core.[Application]
       WHERE   Name = @Softwarename;

       IF ( @Limit IS NULL )
           BEGIN
               SET @Limit = @TotalCount;
           END
       
       SET @End = @Start + @Limit - 1;
   
   
       SET NOCOUNT ON;
       SET TRANSACTION ISOLATION LEVEL READ UNCOMMITTED;
       
       IF ( @IsAscending = 1 )
           BEGIN                
               SELECT  GA.* ,
                       GA.AssetNode.ToString() AS 'AssetPath'
               FROM    ( SELECT    A.AssetId ,
                                   ROW_NUMBER() OVER ( ORDER BY COALESCE(A.OverrideName,
                                                             A.ShortName,
                                                             A.Name) ASC ) AS RowNum
                         FROM      core.Asset AS A
                                   INNER JOIN core.[Application] AS APP ON A.AssetId = APP.AssetId
                         WHERE     APP.Name = @SoftwareName
                                   AND core.IsInExcludedAssetNode(A.AssetNode) = 0
                       ) AS A
                       CROSS APPLY core.GetAsset(A.AssetId) AS GA
               WHERE   A.RowNum BETWEEN @Start AND @End
               ORDER BY A.RowNum;
           END
       ELSE
           BEGIN
               SELECT  GA.* ,
                       GA.AssetNode.ToString() AS 'AssetPath'
               FROM    ( SELECT    A.AssetId ,
                                   ROW_NUMBER() OVER ( ORDER BY COALESCE(A.OverrideName,
                                                             A.ShortName,
                                                             A.Name) DESC ) AS RowNum
                         FROM      core.Asset AS A
                                   INNER JOIN core.[Application] AS APP ON A.AssetId = APP.AssetId
                         WHERE     APP.Name = @SoftwareName
                                   AND core.IsInExcludedAssetNode(A.AssetNode) = 0
                       ) AS A
                       CROSS APPLY core.GetAsset(A.AssetId) AS GA
               WHERE   A.RowNum BETWEEN @Start AND @End
               ORDER BY A.RowNum;
           
           END
   END
GO
